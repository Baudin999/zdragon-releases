Installation

  1. Download the file for your platform
  2. Extract the archive
  3. Run zdragon (or zdragon.exe on Windows)
macOS Users: For the easiest installation, run this in Terminal: 
curl -fsSL https://raw.githubusercontent.com/Baudin999/zdragon-releases/master/install.sh | bash

This installs ZDragon to /Applications and removes the quarantine flag automatically.

Or manually: download, extract, move to Applications, then run xattr -cr /Applications/zdragon.app
